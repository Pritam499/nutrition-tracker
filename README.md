# 🥗 Nutrition Tracker – AI-Powered Health & Diet Monitoring App

🔒 **Note:** This is a showcase repository. The project is available in a below repo for business integration.

---

## 📌 Overview

A full-stack, AI-enabled nutrition tracking system that helps users manage food intake, track health goals, and receive intelligent meal suggestions based on dietary preferences.

Designed for scale, role-based access (Free vs Pro), and built to support future AI and subscription integrations.

---

## ⚙️ Tech Stack

**Frontend:**  
- React.js, Redux Toolkit  
- TailwindCSS 
- Responsive UI & dynamic dashboards

**Backend:**  
- Node.js, Express.js  
- PostgreSQL + Sequelize ORM  
- JWT Auth + Role-base access control

**Extras:**  
- Payment Integration (Razorpay)  
- Deployment-ready (Docker basics)  
- AI support pipeline planned (private)

---

## 🔑 Key Features

- ✅ **User Authentication:** Secure login/signup, JWT-based sessions  
- ✅ **AI Suggestions (Pro):** Smart food recommendations (planned upgrade)  
- ✅ **Role-Based Access:** Feature control based on user type (Free / Pro)  
- ✅ **Food & Nutrition Logs:** Add, edit, and view food history with analytics  
- ✅ **Dashboard:** Daily stats, calorie charts, nutrition breakdown  
- ✅ **Subscription Integration:** Razorpay for Pro features (in progress)
